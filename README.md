# 🛒 E-commerce Frontend

A basic e-commerce frontend built with Vite using routing and component-based architecture. This project includes category pages and a static login UI.

---

## 🚀 Features

- 🧭 Routing for different categories (Men, Women, Kids)
- ⚛️ Built using components and props
- 🔐 Static login page UI

---

## 🛠️ Tech Stack

- Vite
- JavaScript
- React 
- HTML & CSS
